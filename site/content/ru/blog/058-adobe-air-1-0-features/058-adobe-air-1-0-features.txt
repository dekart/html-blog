<p><strong>Возможности Adobe AIR</strong> в кратком содержании:</p>

<ul>
<li><p>Встроенный браузерный движок, позволяющий использовать все возможности HTML, CSS, JavaScript и Flash в едином рабочем пространстве. При разработке AIR-приложений программист одновременно может использовать Javascript для доступа к возможностям Flash, а Flash-программист - к JavaScript-библиотекам. В итоге можно создавать гибридные приложения, сочетающие в себе плюсы обеих платформ.</p></li>
<li><p>Встроенное хранилище данных на базе СУБД SQLite, в котором приложение может хранить пользовательскую информацию. </p></li>
<li><p>Доступ к файловой системе, работа с файлами пользователя</p></li>
<li><p>Поддержка native drag&#8217;n&#8217;drop (перетаскивание данных между приложениями)</p></li>
<li><p>Работа с буфером обмена</p></li>
<li><p>Работа с мультимедиа - звуком, видео (FLV), экраном</p></li>
<li><p>Асинхронная работа с сетевыми ресурсами - поддержка отправки и загрузки данных через AJAX</p></li>

<li><p>Отображение PDF-документов через Adobe Reader</p></li>
<li><p>Управление цифровыми правами на контент (DRM)</p></li>
<li><p>Обновление приложений - программные возможности для приложения самостоятельно обновлять себя</p></li>
<li><p>Поддержка двух операционных систем - Windows и MacOS</p></li>
</ul>
